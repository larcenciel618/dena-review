# dena-review
ターミナルで動く四目並べゲームです。

# Usage
このゲームはターミナル上で動きます。  
以下の2つのコマンドを実行後、石をおきたい列の番号（1から7の数値）を入力することで石を置けます。  
!!一度に256文字以上の文字列は入力しないでください。  
`gcc *.c`  
`./a.out`
